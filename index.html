// 使用教程

01 关键词标签 · 点一下看解释

卡片上的「经典方向 / 新兴活跃」彩色词条可以点：点开大白话解释，再点同一词收起。不会跳转页面。

02 打分功能 · 原理

综合得分 = 你设的三个权重加权：

  • 匹配度：你输入的方向关键词，和老师标签重合越多越高（探索工具，不代表对方愿不愿意带你）
  • 活跃度：是否偏新兴/转向新热点
  • 影响力：Scopus 被引的固定对数刻度（0–100）

拖动滑条会立刻重排卡片。

03 对比功能

在卡片上勾选「对比」（最多 3 位）→ 底部出现对比条 →「查看对比」并排看标签、引用、方向类型。

04 分类功能

结果区 Tab：全部 / 稳健经典 / 稳中拓新 / 快速转向。这是对「方向有没有在变」的定性归类，说明在 Tab 下方的漂移仪图例里。

05 展开完整画像

卡片底部醒目按钮「展开完整画像」→ 四个内页：联系方式 / 自述 / 解读 / 论文。内容区可上下滚动。

06 方向泡泡 · 功能与原理

  • 大小：该方向当前已录入老师人数
  • 颜色：青绿偏经典稳定 → 琥珀偏快速转向（按泡泡内老师类型汇总）
  • 点击：展开该方向老师名单，再点可跳到对应卡片
  • 重叠：跨方向老师会出现在多个泡泡里

07 搜索与清单

  • 按姓名：即时过滤，不记历史
  • 按关键词：参与打分；Enter / 失焦后写入「最近搜索」（本机最多 5 条）
  • 清单（右上角）：把标记为感兴趣/已联系/放弃的老师汇总，可导出 CSV
